# InfraUp Construction Website

A modern, responsive multi-page website built for **InfraUp Construction Company (UAE)**.

## 🌐 Overview

This project is a professional business website designed to showcase construction services, company values, and contact information in a clean and user-friendly layout.

## 🚀 Features

* Fully responsive design (mobile, tablet, desktop)
* Smooth scrolling navigation
* Interactive UI elements (hover effects, animations)
* Multi-page structure (Home, About, Services, Contact)
* Clean and modern design focused on professionalism

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

* `index.html` – Home page
* `about.html` – Company overview
* `services.html` – Services offered
* `contact.html` – Contact form and details
* `styles.css` – Styling
* `script.js` – Interactivity

## 💡 Purpose

This website is developed as a client project for InfraUp Construction Company to establish a strong online presence and showcase their services.
